遊戲概念
========

此迷宮遊戲之背景設定為宇宙探險，主角乘著太空船登陸於其他星球。
此星球地形崎嶇像個迷宮，且不適合人類直接暴露於外，因此主角須全程待在球狀
太空梭裡面。雖然太空梭看似笨重，卻是帶有最新科技以幫助完成任務。

主角的任務為，在此異地星球表面獲取10個重要線索（寶物），但是這10
個東西卻藏在迷宮的各個角落，且由於前述理由主角只能在太空梭轉動行動。
究竟玩家們能不能順利幫助主角完成任務呢？



評分項目：基本
==============

搭建出完整的迷宮，並且使用正確的物理性質(ex: 不會穿牆)
------------------------------------------------------


基本的人物操作
--------------

鍵盤：
- W/S or 上/下：主角往前/後滾
- A/D or 左/右：主角往左/右滾
- Z/C：旋轉方向，影響第一人稱攝影機的視角以及其他按鍵的移動方向
- Ctrl：跳
- Space：剎車


使用光源
--------

場景中有微弱的環境光源，以及一個指向光源。
配合場景故事，這些光源都不會移動。


使用材質與貼圖
--------------

所有Object都有使用。


使用prefab動態生成物件
----------------------

在取得寶物後，寶物的爆炸效果即是使用Prefab動態生成的。


評分項目：進階
==============

使用多個攝影機並且可以互相切換
------------------------------

操作方式：按畫面中右上角的GUI按鍵(Change View)。

視角：
1. 第一人稱視角，跟在主角後面。
2. 俯望視角，跟在主角正上方固定高度處。
3. 綜觀俯望視角，此為可以看到整個地圖的視角，且攝影機不會移動。


小地圖功能
----------

畫面中右上角有個小地圖，其中的綠點便是主角於地圖之所在位置。


迷宮內擺設有功能的機關
----------------------

迷宮內擺設了許多寶物及指示牌，
玩家需蒐集寶物後才能過關。
每次收集到寶物即會增加分數並產生特效。


使用terrain與skybox
-------------------

此地圖使用terrain產生複雜山脈地形，
並用skybox產生宇宙感。


使用particle system
-------------------

1. 在地圖某處有個自然溫泉（會噴煙）。
2. 取得寶物的爆炸特效亦以particle system完成。


加入遊戲的要素
--------------

- 玩家須尋找寶物才能過關，而非單純走到出口。
- 增加了進關的畫面（另一個scene）。
- 加入詭異風格的音效。
